# Coreality Boilerplate

## Introduction
This is a boilerplate for the Coreality projects.

## How to use

### Installation
```zsh
yarn install
# or 
npm install
```

### Run locally
```zsh
yarn dev
# or
npm run dev
```

### Build
```zsh
yarn build
# or
npm run build
```

### Preview build
```zsh
yarn start
# or
npm run start
```
